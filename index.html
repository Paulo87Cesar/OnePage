Wireframe

Wireframe é uma apresentação visual da estrutura básica de uma página. Ele deve ser feito de maneira mais simples possível. como uma espécie de rascunho, sem cores ou imagens. Clique no botão ou escaneie o QR Code para obter o wireframe desta página.
